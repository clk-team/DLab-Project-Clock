# DLab-Project-Clock
本時鐘包含基本的時間顯示，還包刮以下項目：

1.調整時間  
2.分別顯示想觀看的資訊(如：年、月、日)  
3.溫度顯示功能  
4.鬧鐘  
5.碼表  
6.倒計時

其中***顯示時間與調整時間***的功能考慮儒略曆、格里曆，因此星期會自動調校

- 操作及功能介紹請看實際演示影片 
- 原代碼位於n_source中

- **FPGA板為NEXYS 4 DDR**
